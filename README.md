# Kimble

Read real stories by real people

## Installation

```bash
git clone https://github.com/pratikstemkar/kimble
cd kimble
npm i
cd web
npm i
nodemon server.js
cd ..
npm run dev
```

Visit at [http://localhost:3000](http://localhost:3000)
